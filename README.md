🛒 Retail Sales Data Analysis
📌 Project Description

This project analyzes a retail sales dataset to extract meaningful business insights related to:

Customer purchasing behavior

Product category performance

Sales trends

The project demonstrates an end-to-end data analytics workflow, including:

Data cleaning and preprocessing using Python (Pandas)

Exploratory Data Analysis (EDA)

Data visualization using Matplotlib & Seaborn

Loading data into MySQL using SQLAlchemy

Writing SQL queries to answer business questions

Extracting actionable business insights

🎯 Objective

The goal of this project is to simulate how a Data Analyst helps businesses make data-driven decisions using transactional sales data.

🛠 Tools & Technologies Used

Python

Pandas

NumPy

Matplotlib

Seaborn

MySQL

SQLAlchemy

Jupyter Notebook

📂 Dataset Description

The dataset contains retail sales transactions with the following information:

Customer ID

Age

Gender

Product Category

Quantity Purchased

Price Per Unit

Total Purchase Amount

Transaction Date

📊 This dataset allows analysis of:

Customer demographics

Product performance

Purchasing patterns

Revenue trends

🔄 Project Workflow
1️⃣ Data Cleaning (Python)

Data cleaning was performed using Pandas to prepare the dataset for analysis.

Steps performed

Loaded dataset using Pandas

Checked dataset structure and column data types

Handled missing values

Removed duplicate records

Converted date column to datetime format

Feature Engineering

New features created:

transaction_month

transaction_year

Data Quality Checks

Outlier detection using the IQR method

Sales trends over time

📊 Data Visualizations & Insights using Matplotlib
📊 Total Revenue by Product Category
🔎 Business Insight

This chart shows the total revenue generated by each product category.

Key Observations

Electronics generates the highest revenue (~157K)

Clothing follows closely (~156K)

Beauty generates slightly lower revenue (~144K)

Revenue is fairly balanced across all categories

💡 Business Implications

Electronics products likely have higher price points

Clothing products generate revenue through frequent purchases

Businesses should:

Maintain strong inventory for electronics

Promote clothing through seasonal campaigns

Bundle beauty products with clothing promotions

📈 Monthly Sales Trend
🔎 Business Insight

This chart shows how total sales vary across months.

Key Observations

May has the highest sales (~53K)

October and December also show strong performance

September records the lowest sales (~23K)

💡 Business Implications

Increase inventory before high demand months

Run discounts during slow months

Seasonal campaigns can maximize revenue

📊 Distribution of Transaction Amount
🔎 Business Insight

Key Observations

💡 Business Implications

Revenue depends heavily on frequent small purchases

Upselling and bundle deals can increase average order value
